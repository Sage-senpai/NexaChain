# Nexachain Videos

## Required Files

Place your Nexachain introduction video in this folder with the following names:

### Video Files (Required)
- `nexachain-intro.mp4` - Main video file in MP4 format (recommended)
- `nexachain-intro.webm` - Optional: WebM format for better browser compatibility

### Thumbnail (Optional but Recommended)
- `nexachain-thumbnail.jpg` - Video thumbnail/poster image that shows before the video plays

## Video Requirements

### Format
- **Format**: MP4 (H.264) is recommended for best compatibility
- **Resolution**: 1920x1080 (Full HD) or 1280x720 (HD)
- **Aspect Ratio**: 16:9
- **File Size**: Keep under 50MB for faster loading

### Content Suggestions
Your video should cover:
- Brief introduction to Nexachain
- How the investment platform works
- Key benefits and features
- Security and trust factors
- Call to action (sign up/get started)

### Thumbnail Requirements
- **Format**: JPG or PNG
- **Resolution**: 1920x1080 pixels
- **Content**: Should be an engaging frame from your video or a custom design with the Nexachain branding

## Quick Start

1. Add your video file(s) to this folder:
   - `nexachain-intro.mp4` (required)
   - `nexachain-intro.webm` (optional)

2. Add a thumbnail image (optional):
   - `nexachain-thumbnail.jpg`

3. The video player will automatically detect and use these files

4. Push to GitHub and your video will be live!

## Testing Locally

Before pushing to production:
1. Place your video files in this folder
2. Run `npm run dev`
3. Check the landing page (/) and FAQ page (/faq)
4. Verify the video plays correctly and looks good

## Fallback

If no video file is found, users will see a "Coming Soon" message.
